# breakeven.md

## Unit Economics & Break-even Analysis

### Cost per Active User
- **Compute**: $5/user/month
- **Storage**: $2/user/month
- **Bandwidth**: $1/user/month  
**Total Cost per Active User**: **$8/user/month**

### Pricing Tiers
| Tier         | Price ($/mo) | Features                                                                 |
|--------------|--------------|--------------------------------------------------------------------------|
| Basic        | $20          | Basic task management, ERP integration, and reporting tools             |
| Pro          | $50          | All Basic features + advanced analytics, user roles, and permissions     |
| Enterprise    | $100         | All Pro features + dedicated support, custom integrations, and SLA       |

### Customer Acquisition Cost (CAC) Range
- Estimated CAC: **$100 - $200** per user

### Lifetime Value (LTV) Estimate
- Average Monthly Revenue per User (ARPU): $50 (assuming mix of Basic and Pro tiers)
- Average Customer Lifespan: 24 months  
**LTV**: **$50 * 24 = $1,200**

### Break-even Users Count
- Fixed Costs (monthly): $5,000 (assumed for operations, marketing, etc.)
- Contribution Margin per User (ARPU - Cost per Active User): $50 - $8 = $42  
**Break-even Users Count**:  
\[ \text{Break-even Users} = \frac{\text{Fixed Costs}}{\text{Contribution Margin per User}} = \frac{5000}{42} \approx 119 \text{ users} \]

### Path to $10K MRR
- Target Monthly Recurring Revenue (MRR): $10,000
- Average Revenue per User (ARPU): $50 (assuming a mix of tiers)

**Required Users**:  
\[ \text{Required Users} = \frac{\text{Target MRR}}{\text{ARPU}} = \frac{10000}{50} = 200 \text{ users} \]

**Tier Strategy**:
- If targeting **Pro Tier** ($50/user/month):
  - **Users Needed**: 200 users
- If targeting **Enterprise Tier** ($100/user/month):
  - **Users Needed**: 100 users

**Conclusion**: To achieve $10K MRR, we can either aim for 200 users on the Pro tier or 100 users on the Enterprise tier.
